Tiny Timer

A small portable countdown timer

View the Project on GitHub matarz/tiny_timer

  • Download ZIP File
  • Download TAR Ball
  • Fork On GitHub

This is a small simple portable countdown timer. When you run it, a prompt will pop out asking for the length of the countdown in minutes. It will then display a tooltip at the upper left corner (you can move it by clicking F2) showing the countdown.

After countdown reaches 0, it will warn you by displaying a message box and playing a sound.

The language used here is AutoIt. For instructions on compiling .au3 files see here.
